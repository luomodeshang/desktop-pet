# 桌面宠物程序 - Windows小白用户完全指南

## 🎯 目标用户
- 完全不懂编程的普通Windows用户
- 没有Python环境的用户
- 不想折腾技术细节的用户
- 只想简单双击就能用的用户

## 📦 三种获取方式（任选其一）

### 方式1：一键获取器（最简单！）
1. 下载 `Windows_一键获取器.bat`
2. 双击运行
3. 按照提示操作
4. 程序自动完成所有安装步骤

### 方式2：小白专用安装器
1. 下载 `Windows_小白专用安装器.bat`
2. 双击运行
3. 自动安装Python和所有依赖
4. 自动创建快捷方式

### 方式3：完整安装助手
1. 下载 `Windows_完整安装助手.bat`
2. 以管理员身份运行
3. 提供最完整的安装选项
4. 包含备份和恢复功能

## 🚀 快速开始（推荐方式1）

### 第一步：下载
1. 访问项目页面：https://github.com/luomodeshang/desktop-pet
2. 找到并下载 `Windows_一键获取器.bat`

### 第二步：运行
1. 双击下载的 `Windows_一键获取器.bat` 文件
2. 如果出现安全警告，点击"更多信息" → "仍要运行"
3. 按照屏幕提示操作

### 第三步：使用
1. 安装完成后，桌面上会出现"桌面宠物"图标
2. 双击图标启动程序
3. 首次运行选择一张照片
4. 开始享受你的桌面宠物！

## 🐾 程序功能

### 基本操作
- **选择照片**：首次运行选择一张照片，程序会自动生成动漫风格宠物
- **右键点击**：投喂食物（汉堡🍔、炸鸡🍗、螺蛳粉🍜）
- **左键点击**：触发动作（挥手👋、跳跃🤸、踱步🚶）
- **自动活动**：宠物会自己切换状态（站立→蹲下→睡眠）

### 高级功能
- **状态保存**：退出程序时自动保存宠物状态
- **自动更新**：程序会自动检查更新
- **多宠物支持**：可以创建多个不同的宠物
- **自定义食物**：可以添加自己喜欢的食物

## ⚠️ 常见问题解决

### 问题1：杀毒软件误报
```
解决方法：
1. 打开杀毒软件
2. 找到"信任列表"或"排除列表"
3. 添加桌面宠物程序文件夹
4. 或暂时关闭杀毒软件安装
```

### 问题2：Python安装失败
```
解决方法：
1. 手动下载Python：https://www.python.org/downloads/
2. 下载Python 3.10版本
3. 安装时务必勾选"Add Python to PATH"
4. 重新运行安装器
```

### 问题3：程序无法启动
```
解决方法：
1. 检查是否安装了Python
2. 尝试以管理员身份运行
3. 检查网络连接（首次安装需要）
4. 查看错误提示并搜索解决方案
```

### 问题4：找不到下载文件
```
解决方法：
1. 直接访问：https://github.com/luomodeshang/desktop-pet
2. 点击"Code" → "Download ZIP"
3. 解压后找到bat文件运行
```

## 🔧 技术需求

### 最低系统要求
- **操作系统**：Windows 7/8/10/11（64位）

- **操作系统**：Windows 10/11
- **内存**：4GB RAM或更多
- **磁盘空间**：500MB可用空间
- **显卡**：支持OpenGL 2.0

## 🎨 自定义设置

### 修改宠物外观
1. 准备一张自己的照片
2. 运行程序时选择新照片
3. 程序会自动生成动漫风格宠物

### 添加自定义食物
1. 打开 `config/config.json`
2. 找到"foods"部分
3. 添加新的食物项
4. 保存并重启程序

### 修改宠物行为
1. 打开 `config/config.json`
2. 修改动作参数
3. 调整状态切换时间
4. 保存并重启程序

## ⏰ 日常使用技巧

### 最佳实践
1. **定期备份**：备份 `config` 文件夹保存设置
2. **清理缓存**：定期清理 `temp` 文件夹
3. **更新检查**：每月检查一次更新

2. 确保足够的磁盘空间
3. 定期重启程序释放内存
4. 使用SSD硬盘获得更好体验
