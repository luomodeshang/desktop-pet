桌面宠物项目已上传到GitHub

项目地址: https://github.com/luomodeshang/desktop-pet

项目功能

  • 🖼️ 照片转动漫头像
  • 🐾 多种宠物状态
  • 🍔 预设食物: 汉堡、炸鸡、螺蛳粉
  • 🎮 交互功能
  • 💾 自动状态保存

快速开始

# 安装依赖
pip install -r requirements.txt

# 处理照片
python process_photo_simple.py 你的照片.jpg

# 运行程序
python src/main.py
