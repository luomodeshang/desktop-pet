# Desktop Pet v3.0 - Q版重生

## 重大更新

### 1. Q版小人生成器 (全新增)
从照片生成属于你自己的 Q 版卡通角色！
- 人脸检测 → 提取面部特征（肤色、发色、瞳色、唇色）
- 自动生成大头小身体 Q 版比例
- 大眼睛 + 双高光 + 腮红（日式卡通风格）
- 支持正面、半侧面照片

### 2. 衣服颜色自动匹配 (全新)
- 从照片身体区域提取衣服颜色（主色+副色）
- Q 版角色自动穿上同色衣服
- V领装饰使用副色搭配

### 3. Live2D 风格动画引擎 (全新)
角色会像 Live2D 一样生动——纯 QPainter 实现，无需额外依赖！
- 呼吸：身体微微起伏 + 头部上下（持续循环）
- 眨眼：随机 3~5 秒一次，流畅 6 帧过渡
- 头发飘动：主发丝 + 刘海 + 侧发同步摆动
- 视线移动：眼睛缓慢左右扫视

### 4. 情绪驱动表情系统 (全新)
角色会根据你的操作做出不同表情：
| 触发 | 表情 |
|------|------|
| 左键点击 | 开心微笑 + 眯眼 + 脸红 |
| 双击 | 惊喜张嘴 + 挑眉 + 瞪眼 |
| 投喂 | 嘴巴规律咀嚼 |
| 空闲→蹲下 | 半闭眼，放松 |
| 空闲→睡眠 | 全闭眼 |

### 5. 交互增强
- 双击触发旋转跳跃
- 托盘菜单添加宠物大小调节（小/中/大/超大）
- 配置自动持久化

### 6. 性能优化
- 动画引擎从 5 个独立 QTimer → 1 个 30FPS 主循环
- 动画帧预渲染缓存（无需每帧重复加载图像）
- 状态管理从字符串 → Enum，更稳定

## 依赖

与 v2.0 完全相同：
- Python 3.8+
- PyQt5
- opencv-python
- Pillow (PIL)
- NumPy

## 已知限制

- 照片需包含清晰正面/半侧面人脸
- 光线充足时效果最佳
- 复杂背景可能影响衣服颜色提取
- Live2D 风格动画为纯 QPainter 实现，非真实 Live2D 模型（但更轻量、零依赖）
