## 🚀 安装与运行

### 方案A：使用单个EXE文件（最简单）
1. **下载** `DesktopPet.exe`
2. **双击运行**即可
3. **首次运行**会提示选择照片
4. **宠物出现**在桌面右下角

### 方案B：使用绿色免安装版
1. **下载** `DesktopPet_Windows_v1.0.2.zip`
2. **解压**到任意文件夹（如：`D:\桌面宠物`）
3. 进入解压后的文件夹
4. **双击** `DesktopPet.exe`

### 方案C：使用安装程序（最规范）
1. **下载** `DesktopPet_Setup.exe`
2. **双击运行**安装向导
3. 选择安装目录（默认：`C:\Program Files\桌面宠物程序`）
4. 选择是否创建桌面快捷方式
5. 点击"安装"完成
6. 从开始菜单或桌面快捷方式启动

## 🎮 使用教程

### 第一步：选择照片
1. 程序启动后，弹出文件选择窗口
2. 选择一张你的照片（支持JPG/PNG）
3. 程序自动转换为动漫风格宠物
4. 转换时间：约10-30秒

### 第二步：基本操作
- **左键点击宠物**：触发随机动作（挥手、跳跃、踱步）
- **右键点击宠物**：弹出投喂菜单
- **拖动宠物**：移动到桌面任意位置
- **系统托盘图标**：右键可退出或设置

### 第三步：投喂互动
右键菜单提供三种食物：
1. **🍔 汉堡** - 宠物开心吃汉堡
2. **🍗 炸鸡** - 宠物兴奋吃炸鸡  
3. **🍜 螺蛳粉** - 宠物好奇尝试螺蛳粉

### 第四步：智能状态
- **活跃状态**：宠物站立，等待互动
- **休息状态**：2分钟无互动 → 蹲下
- **睡眠状态**：5分钟无互动 → 睡觉
- **点击唤醒**：点击睡觉的宠物恢复活跃

## ⚙️ 配置与自定义

### 配置文件位置
```
C:\Users\<你的用户名>\AppData\Roaming\桌面宠物程序\data\
```

### 主要配置文件
1. **pet_state.json** - 宠物状态和位置
2. **user_settings.json** - 用户设置
3. **photo_cache/** - 照片缓存目录

### 自定义设置示例
编辑 `user_settings.json`：
```json
{
  "pet_size": 100,           // 宠物大小百分比（50-200）
  "auto_sleep_minutes": 10,  // 自动睡觉时间（分钟）
  "food_effects": true,      // 食物特效开关
  "sound_effects": false,    // 音效开关（暂无）
  "start_minimized": false   // 启动时最小化
}
```

## 🐛 常见问题解决

### 问题1：程序无法启动
**症状**：双击没反应，或闪退

**解决**：
1. **以管理员身份运行**：右键 → 以管理员身份运行
2. **安装运行库**：下载安装 [VC++运行库](https://aka.ms/vs/17/release/vc_redist.x64.exe)
3. **兼容性模式**：右键 → 属性 → 兼容性 → Windows 8
4. **更新系统**：Windows更新到最新版本

### 问题2：照片转换失败
**症状**：提示"无法识别人脸"或转换后是黑块

**解决**：
1. **使用正面清晰照片**：正脸，光线充足
2. **照片尺寸合适**：500x500像素以上，不超过5MB
3. **格式正确**：JPG或PNG格式
4. **重新选择**：关闭程序重新运行选择照片

### 问题3：宠物显示异常
**症状**：宠物显示为黑色方块或花屏

**解决**：
1. **更新显卡驱动**：访问显卡官网下载最新驱动
2. **关闭硬件加速**：编辑 `user_settings.json`，添加 `"hardware_acceleration": false`
3. **重启程序**：完全退出后重新启动

### 问题4：杀毒软件误删
**症状**：程序被隔离或删除

**解决**：
1. **恢复文件**：在杀毒软件隔离区恢复
2. **添加信任**：将程序目录添加到杀毒软件白名单
3. **暂时关闭**：运行程序时暂时关闭实时保护

## 🔄 更新与卸载

### 检查更新
程序启动时自动检查，也可手动：
1. 右键系统托盘图标
2. 选择"检查更新"
3. 如有新版本，提示下载

### 完全卸载
#### 方法一：控制面板
1. 控制面板 → 程序和功能
2. 找到"桌面宠物程序"
3. 点击"卸载"

#### 方法二：安装目录
1. 打开程序安装目录
2. 运行 `unins000.exe`
3. 按向导卸载

#### 保留用户数据
卸载时选择"保留我的数据"，下次安装可恢复宠物状态。
