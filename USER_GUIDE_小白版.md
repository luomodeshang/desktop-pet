# 🐱 桌面宠物程序 - 小白使用指南

## 🎯 一句话说明
**把你的照片变成会动的桌面宠物，可以喂食、互动、陪你工作！**

## 📥 如何下载安装

### 方法一：一键安装（推荐）
1. **点击下载**：[DesktopPet_Setup.exe](https://github.com/luomodeshang/desktop-pet/releases/latest/download/DesktopPet_Setup.exe)
2. **双击运行**安装程序
3. **下一步 → 下一步**完成安装
4. **桌面**上会出现宠物图标

### 方法二：绿色免安装
1. **点击下载**：[DesktopPet_Windows_v1.0.0.zip](https://github.com/luomodeshang/desktop-pet/releases/latest/download/DesktopPet_Windows_v1.0.0.zip)
2. **解压**到任意文件夹
3. **双击** `DesktopPet.exe` 运行

> 💡 **提示**：如果杀毒软件报警，请点击"允许"或"信任此程序"

## 🚀 第一次使用

### 1. 选择你的照片
![选择照片](https://via.placeholder.com/400x200/4CAF50/FFFFFF?text=选择一张你的照片)

程序启动后会让你选择一张照片：
- 可以是**自拍**、**宠物照**、**动漫头像**
- 支持 JPG、PNG 格式
- 建议选择**正面清晰**的照片

### 2. 等待转换
![转换中](https://via.placeholder.com/400x200/2196F3/FFFFFF?text=正在转换为动漫风格...)

程序会自动：
- 识别照片中的人脸
- 转换为**动漫风格**
- 生成可爱的宠物形象

### 3. 开始互动！
![宠物出现](https://via.placeholder.com/400x200/FF9800/FFFFFF?text=你的桌面宠物来啦！)

宠物会出现在桌面右下角，现在可以开始玩了！

## 🎮 怎么玩？

### 基本操作
| 操作 | 效果 | 说明 |
|------|------|------|
| **左键点击** | 触发动作 | 挥手、跳跃、踱步 |
| **右键点击** | 投喂食物 | 汉堡、炸鸡、螺蛳粉 |
| **拖动宠物** | 移动位置 | 放到你喜欢的位置 |
| **双击宠物** | 切换状态 | 站立、蹲下、睡觉 |

### 宠物状态
宠物会自动切换状态：
1. **站立** (默认) - 正常状态
2. **蹲下** (闲置5分钟) - 有点累了
3. **睡觉** (闲置10分钟) - Zzz...

> 🕒 **时间**：长时间不动会自动睡觉，点击一下就会醒来！

### 食物系统
右键点击宠物可以选择三种食物：
- **🍔 汉堡** - 普通食物
- **🍗 炸鸡** - 美味食物  
- **🍜 螺蛳粉** - 特色食物

宠物吃完会有不同的反应动画！

## ⚙️ 个性化设置

### 修改宠物形象
如果想换照片：
1. 右键点击系统托盘图标
2. 选择"更换照片"
3. 选择新照片

### 调整大小
宠物默认大小适中，可以调整：
1. 右键系统托盘图标
2. 选择"设置"
3. 拖动滑块调整大小

### 开机自启动
想让宠物每次开机都出现？
1. 右键系统托盘图标
2. 选择"开机自启动"
3. 勾选"随Windows启动"

## ❓ 常见问题

### Q1: 程序打不开怎么办？
**A:** 尝试以下方法：
1. 以**管理员身份**运行
2. 安装 [VC++运行库](https://aka.ms/vs/17/release/vc_redist.x64.exe)
3. 关闭杀毒软件再试

### Q2: 照片转换失败？
**A:** 可能是照片问题：
1. 换一张**正面清晰**的照片
2. 确保照片中**有人脸**
3. 照片不要太大（建议2MB以内）

### Q3: 宠物不动了？
**A:** 可能是：
1. 宠物在**睡觉** - 点击一下唤醒
2. 程序**卡住**了 - 重启程序
3. 系统**资源不足** - 关闭一些程序

### Q4: 如何卸载？
**A:** 两种方法：
1. **控制面板** → 程序和功能 → 卸载"桌面宠物程序"
2. 运行安装目录下的 `unins000.exe`

## 🎨 高级玩法

### 自定义动作
技术用户可以修改配置文件：
1. 打开 `config/settings.json`
2. 修改动作参数
3. 保存重启

### 添加新食物
1. 准备食物图片（PNG透明背景）
2. 放到 `assets/images/foods/`
3. 修改 `config/settings.json`

### 开发模式
如果你是开发者：
1. 安装 Python 3.8+
2. 安装依赖：`pip install -r requirements.txt`
3. 运行：`python src/main.py`

## 📱 多设备同步

### 备份宠物数据
宠物数据保存在：
```
C:\Users\你的用户名\AppData\Roaming\桌面宠物程序\data\
```

可以备份：
- `pet_state.json` - 宠物状态
- `user_settings.json` - 用户设置
- `photo_cache/` - 照片缓存

### 恢复到新电脑
1. 在新电脑安装程序
2. 复制备份文件到相同位置
3. 启动程序，宠物就回来啦！
